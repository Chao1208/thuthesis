## 图表生成方法

### 方法一：使用Python脚本生成（推荐）

1. **安装依赖**（如果尚未安装）：
```bash
pip3 install matplotlib numpy
```

2. **运行脚本生成图表**：
```bash
cd /Users/lichao/Documents/project/github/thuthesis/myfigure
python3 generate_charts.py
```

3. **结果**：将在当前目录生成4个PDF文件：
   - internet-development-2000-2010.pdf
   - rnd-investment-growth.pdf
   - agile-adoption-rate.pdf
   - innovation-incentive-system.pdf

### 方法二：使用在线工具生成

如果无法安装Python库，可以使用在线工具：

1. **Plotly Chart Studio** (https://chart-studio.plotly.com/)
2. **DataWrapper** (https://www.datawrapper.de/)
3. **Flourish** (https://flourish.studio/)

按照 `README-图表说明.md` 中的数据要求制作图表，导出为PDF格式。

### 方法三：使用Excel/Numbers制作

1. 打开Excel或Numbers
2. 根据 `README-图表说明.md` 输入数据
3. 创建相应类型的图表
4. 调整样式使其符合学术规范
5. 导出为PDF格式（推荐）或高分辨率PNG（至少300 DPI）

## 启用图表的步骤

图表文件生成后，需要在LaTeX中启用它们：

1. 打开 `mydata/chap03.tex` 文件

2. 找到类似这样的注释行：
```latex
% 注释：此处应插入图片文件 internet-development-2000-2010.pdf
% \includegraphics[width=0.85\textwidth]{myfigure/internet-development-2000-2010.pdf}
```

3. 取消注释 `\includegraphics` 行（删除前面的 `%`）：
```latex
% 注释：此处应插入图片文件 internet-development-2000-2010.pdf
\includegraphics[width=0.85\textwidth]{myfigure/internet-development-2000-2010.pdf}
```

4. 对所有4个图都进行同样的操作

5. 重新编译论文：
```bash
cd /Users/lichao/Documents/project/github/thuthesis
make clean
make
```
