## 图表制作建议

### 工具推荐
1. **Python + Matplotlib/Seaborn**: 适合数据驱动的图表制作
2. **R + ggplot2**: 适合统计图表
3. **Excel/Numbers**: 适合简单图表
4. **专业工具**: Origin, GraphPad Prism（科研专用）
5. **在线工具**: Plotly, ECharts

### 设计规范
- **字体**: 使用无衬线字体（如Arial, Helvetica）
- **字号**: 标题12pt，坐标轴10pt，标签9pt
- **颜色**: 使用色盲友好的配色方案
- **分辨率**: 至少300 DPI（用于印刷）
- **格式**: PDF矢量格式（保证缩放不失真）
- **大小**: 宽度适配论文版面（通常为12-15cm）

### LaTeX集成
所有图表已在chap03.tex中预留位置，使用以下格式：
```latex
\begin{figure}[htbp]
\centering
\includegraphics[width=0.85\textwidth]{myfigure/filename.pdf}
\caption{图标题}
\label{fig:label-name}
\end{figure}
```

1. 根据本文档的数据要求准备数据
2. 选择合适的工具制作图表
3. 按照设计规范调整图表样式

7. 编译论文查看效果

## 注意事项

- 所有图表必须是矢量格式（PDF），不要使用位图（PNG, JPG）
- 图表中的文字大小要适中，确保缩小后仍然清晰可读
- 颜色使用要统一，建议使用同一套配色方案
- 图表的标题和标签要使用中文
- 数据标签要准确，小数点后保留1-2位即可
- 如果图表过于复杂，考虑分解为多个子图

## 快速生成图表的Python示例代码

```python
import matplotlib.pyplot as plt
import numpy as np

# 设置中文字体
plt.rcParams['font.sans-serif'] = ['Arial Unicode MS']
plt.rcParams['axes.unicode_minus'] = False

# 示例：生成图3.1
years = [2000, 2005, 2010]
netizens = [0.23, 1.11, 4.57]
companies = [0.1, 0.5, 2.0]

fig, ax1 = plt.subplots(figsize=(10, 6))
ax2 = ax1.twinx()

ax1.plot(years, netizens, 'o-', color='#2E86AB', linewidth=2, markersize=8, label='网民规模')
ax2.plot(years, companies, 's-', color='#A23B72', linewidth=2, markersize=8, label='企业数量')

ax1.set_xlabel('年份', fontsize=12)
ax1.set_ylabel('网民规模（亿人）', fontsize=12, color='#2E86AB')
ax2.set_ylabel('互联网企业数量（万家）', fontsize=12, color='#A23B72')

plt.title('中国互联网网民规模与企业数量增长趋势（2000-2010）', fontsize=14)
ax1.legend(loc='upper left')
ax2.legend(loc='upper right')

plt.grid(True, alpha=0.3)
plt.tight_layout()
plt.savefig('internet-development-2000-2010.pdf', dpi=300, bbox_inches='tight')
plt.close()
```
